1. 개요

GVT 소형 물탱크차는 원심펌프 기반의 고성능 소방자동차로, 일반 화재는 물론 유류 화재 대응까지 고려되어 제작되었습니다. KFI(한국소방산업기술원) 인정 사양이며, 고기동 샤시에 4,500L 물탱크와 각종 방수·흡수 장비가 통합되어 있습니다.

핵심 포인트 — 엔진+기어박스 PTO로 펌프를 구동하며, 전자클러치 및 지수/역류방지 밸브로 빠른 양수·방수를 지원합니다.

3. 관련 법규·검사

최대 안전 경사각(안전검사)

  • 승용·화물·특수자동차(연결차 포함): 공차 상태에서 35도 (단, 총중량이 차량중량의 1.2배 이하이면 30도)
  • 승합 11인 이상: 적차 상태에서 28도

※ 안전검사 시 좌·우 경사대에서 전복 없이 요구 경사각을 만족해야 합니다.

KFI 인정

본 차량의 소방장비는 한국소방산업기술원(KFI) 인정 대상 구성품을 포함합니다. 현장 배치 전 수검·등록 절차를 완료하십시오.

4. 외관 & 조작반 위치

차량 외부에는 V자형 경광등, 교차등, 후방 카메라, 후부사다리, 측·후면 작업등, 흡수관함, 펌프룸 등이 배치됩니다. 펌프룸 조작반에는 압력계/연성계/전자식 수량계/중계구/방수구/보수구/흡수구/냉각수 밸브/부동액 탱크·밸브/자체급수/비상 RPM 등이 집약돼 있어 신속한 원스톱 조작이 가능합니다.

팁 조작 전 메인 전원을 켜고, 사용할 방수구/밸브만 개방하여 불필요한 압력 손실을 막습니다.

5. 펌프 구조·기능

요약

  • 주 펌프(임펠러): 원심력으로 흡·토출. 임펠러 형상은 성능 등급에 따라 치수가 다름.
  • 진공펌프: 흡수 초기 수로 내 공기를 배기하여 양수를 보조. 연속 60초 이상 가동 금지(과열·파손 위험)
  • 역류방지밸브: 진공 중 주펌프 내부로 공기 유입 차단, 방수 완료 후 펌프 기밀 유지.
  • 지수밸브: 진공 시 다이아프램이 하강하며 개방, 양수 완료 후 진공펌프로의 역류 차단.
  • 냉각계통: PTO 오일 과열 방지를 위해 펌프수로 보조 냉각, 스트레이너 주기 세척.
주의 물이 없는 상태(공회전)로 펌프를 돌리지 마십시오. 내부 가스켓 손상 위험이 큽니다.

6. 운용 절차 (급수·방사·종료)

6.1 물탱크 급수

맨홀 급수

  1. 급수에 용이한 위치로 정차 → 맨홀 손잡이를 좌로 돌려 커버 개방.
  2. 급수전 호스 연결 후 만수위까지 급수 → 맨홀 닫음.

보수구(소화전) 급수

  1. 소화전–보수구 호스 연결 → 소화전 및 보수구 밸브 열림.
  2. 만수위 도달 시 보수구 밸브 닫힘 → 소화전 닫고 호스 해체 → 보수구 드레인.

자체급수(저수지 등 외수원)

  1. 흡수관을 흡수구에 연결 후 스트레이너를 수중 깊게 잠김.
  2. 모든 밸브 닫힘 → 흡수밸브 열림 → 브레이크 고정 후 P.T.O ON.
  3. 진공펌프(양수장치) 작동 → 유입 시작되면 자동 정지. 엔진 1,200–1,500rpm 이내 유지.
  4. 자체급수 밸브 열림 → 수량계 만수위까지 급수 → 자체급수 닫힘.

6.2 화재 시 방사

  1. 현장 정차·주차브레이크·중립 → 메인 전원 ON → P.T.O 버튼 3초간 눌러 ON.
  2. 메인 밸브 열림 → 압력 형성 확인 → 사용할 방수구 열림.
  3. 스로틀로 방수압 조절(관창/호스 상태를 보며 과압 주의). 야간에는 펌프실 조명 점등.

중계 방사

타 차량의 방수구 ↔ 우리 차량 중계구(65A) 직결. 흡수구는 반드시 닫힘. 연성계로 중계압 확인.

저수지 활용 방사

저수지에서 자체급수로 만수위까지 채운 뒤, 일반 방사 절차와 동일하게 진행합니다.

6.3 종료 후 조치

  • 배관/탱크 잔수 완전 배수 → 모든 밸브 열림(부동액 밸브 제외) → 약 5분 후 닫음.
  • 동절기: 지수변 체크밸브 눌러 잔수 제거, 부동액 주입 절차 수행(약 1L).
  • 진공펌프 오일·PTO/펌프 기어오일 잔량 확인·보충.

7. 운행/사용 시 유의사항

  • 조작 전 매뉴얼 숙지·점검. 임의 개조/봉인 훼손/압력 조정 금지.
  • 음주·수면부족 상태 운전/조작 금지. 엔진 가동 중 구동부(회전부) 접근 금지.
  • PTO 작동 시 클러치 밟은 뒤 약 5초 경과 후 서서히 연결(급작동 시 기어 마모·파손).
  • 진공펌프 연속 60초 초과 가동 금지(과열·파손). 흡수호스는 완전 침수·소용돌이 방지.
  • 스트레이너·냉각수 스트레이너 주기 세척. 냉각수 밸브로 PTO 과열 방지.
  • 펌프 공회전 금지(무수 상태 방전). 방수 시 사람·취약 구조물을 향한 직사 금지.
  • 맨홀 사용 후 뚜껑 닫힘 재확인(주행 중 슬로싱·유출 위험).
  • 겨울철 사용 후 드레인 미실시 시 동파 위험 → 즉시 배수·부동액 주입.

8. 흔히 하는 실수와 조치법

증상가능 원인조치
진공이 안 걸림/양수 지연 오일 부족, 윤활배관 느슨함/막힘, 흡수호스 누수·패킹 불량, 반대측 흡수 볼밸브 개방, 역류방지/지수밸브 불량, 배관 느슨함, 흡·토출 높이차 과다 오일·배관 상태 점검/보충, 불사용측 흡수밸브 닫힘, 밸브 이물 제거·작동 확인, 수심 깊은 수원지 재선정
연성계 진공 과상승 스트레이너 진흙·잡초 막힘, 바닥에 묻힘, 호스 내부 고무판 탈락 스트레이너 청소·위치 조정, 호스 교체
방수압 저하/급락 임펠러 손상·이물 막힘, 흡수원 과고/중계호스 과장거리, 흡입측 공기누출 엔진 저속·정지 후 원인 점검, 접속부 재체결, 필요 시 급수량 증대·방수량 감소로 압력 0 이상 유지
작동 후 잔수 잔존 배수 밸브 미개방·시간 부족 주행 중 흔들림 이용하여 배수 완료 → 격납 전 밸브 닫음

9. 점검·정비

일상 점검

  • 배수 밸브 전부 개방해 잔수 완전 배수(복귀 중 흔들림 이용) → 격납 전 모두 닫음.
  • 외부 수분 제거·비도장부 기름칠. 엔진오일/연료/PTO·진공·펌프 기어오일 보충.
  • 파이프 느슨함, 전기 터미널·배선 점검. 흡수/방수 호스 세척 후 응달 건조.

주간 점검(예)

  • PTO 해제 시 배관 배수 상태 확인, PTO 오일 누유·적정량 확인.
  • 진공펌프 가동 상태·소음 점검, 지수밸브·역류방지밸브 작동 점검.
  • 스로틀·자동개폐밸브·게이지 정상 동작 확인, 배관 누수·보수구 역류 여부 점검.

동절기 유의

  • 화재 종료 즉시 드레인 → 진공 작동으로 -200 ~ -300mmHg 형성 → 물펌프 부동액 밸브 개방, 레벨게이지 1/4까지 주입(약 1L) → 닫음.
  • PTO를 5–10초 구동 후 OFF, 부동액 밸브 닫힘 재확인.

10. 에어시스템 & 캡틸팅

에어시스템

메인 에어솔레노이드 밸브는 특장부 에어 차단·공급을 제어합니다. 셀렉터 스위치를 11시 방향으로 두면 특장 전원 ON/OFF에 연동, 1시 방향은 비상 시 특장 전원과 무관하게 에어 공급이 가능합니다.

캡 틸팅

  1. 주변 안전 확인 → 틸팅 버튼 → 오일 확인 후 틸트키 해제.
  2. 커버 열고 ON으로 올리면 자동 상승·정지. 하강 시 노브를 하강 위치로 고정 후 ON.
  3. 상승 시 안전바+볼트 체결, 정비 완료 후 반드시 제거·하강.

11. 주요 위험/사고 유형(요약)

※ 아래 항목은 매뉴얼 경고·주의사항을 바탕으로 정리한 “사고유형/위험요인”이며, 개별 실제 사고사례 DB는 추후 연계 가능합니다.

  • 경사각 초과 정차·작업 → 전복 위험. 검사 기준(35° 등) 내 운영 필요.
  • 진공펌프 과열 파손 → 60초 이상 연속가동, 오일 부족 시 발생. 가동시간 엄수·오일 점검.
  • 동절기 동파 → 드레인 미흡·부동액 미주입. 즉시 배수·부동액 주입.
  • 맨홀 미잠금 → 주행 중 슬로싱으로 유출·낙상 위험. 출발 전 재확인.
  • 흡수라인 공기 유입 → 캐비테이션·압력 급락. 스트레이너 완전 침수·연결부 재체결.
  • 방수 방향·압력 미관리 → 인명·시설 피해. 관창자와 압력·지형 소통 필수.

13. 부록

방수량 계산식

Q = 0.653 × D² × √P ÷ 1000 (Q: L/min, D: 노즐 직경 mm, P: kg/㎠)

저수지 선정 기준(요약)

  • 현장과 가까울수록 유리(마찰·연결시간 최소화).
  • 펌프 중심–수면 간 거리는 가까울수록 양수 빠름.
  • 수심 깊고 수량 많은 장소(바닥 접촉·소용돌이·공기혼입 방지).
  • 깨끗한 수원지(스트레이너 막힘 방지), 가스 발생 지역 회피.
  • 먼지·쓰레기·모래 많은 수원지 사용 지양(펌프 손상 위험).

용어

  • 연성계: 진공/압력 겸용 게이지. 흡수 시 진공(-)지시, 중계 시 압력(+)지시.
  • 보수구: 소화전 등 외부 급수로 탱크 보급 시 사용하는 65A 라인.
  • 자체급수: 외부 수원에서 탱크로 직접 양수·보급하는 기능.
